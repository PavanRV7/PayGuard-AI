## 🌟 Overview

PayGuard AI is an AI-powered financial fraud detection and intelligence platform developed using Machine Learning, Python, and Power BI.

The system analyzes transaction behavior, predicts fraudulent activities using a Random Forest model, assigns AI-driven risk scores, and visualizes fraud intelligence through interactive enterprise-grade dashboards.

This project combines:

* Machine Learning
* Risk Scoring
* Fraud Intelligence Analytics
* Real-Time Monitoring Dashboards
* Executive Reporting

---

# ❗ Problem Statement

Financial fraud causes billions of dollars in losses globally every year.

Traditional rule-based systems struggle to:

* detect evolving fraud patterns,
* prioritize suspicious transactions,
* provide real-time fraud intelligence.

PayGuard AI addresses these challenges by:

* identifying fraudulent transactions using AI,
* assigning dynamic fraud risk scores,
* categorizing transaction risk levels,
* enabling fraud investigation workflows,
* delivering executive-level fraud intelligence dashboards.

# ⚙️ Machine Learning Workflow

## 1. Data Preprocessing

* Missing value handling
* Feature encoding
* Fraud-specific feature engineering
* Balance error calculations

## 2. Feature Engineering

Created important fraud intelligence features:

* balanceErrorOrig
* balanceErrorDest
* transaction type encoding
* risk scoring variables

## 3. Handling Imbalanced Data

Used:

```python
SMOTE
```

to balance fraudulent and genuine transactions.

## 4. Model Training

Implemented:

* Random Forest Classifier
* Logistic Regression (comparison)

## 5. Risk Scoring

Generated:

* Fraud probabilities
* AI risk levels
* High-risk alerts
